# NU-IDE

A lightweight and simple IDE built with Qt for Arduino development.

## Features

* Basic code editing
* Simple and clean interface

## Requirements

* Qt
* Arduino CLI or Arduino IDE (for compilation/upload)

## Installation

1. Clone or download this repository
2. Open the project in Qt Creator
3. Build the project
4. Run the executable

## Usage

1. Open or create an Arduino sketch (`.ino` file)
2. Edit your code in the editor
3. Compile and upload code

## Notes

* Make sure Arduino CLI or IDE is properly installed and added to your system PATH
* This IDE is minimal and intended for simple workflows with very little memory and storage usage
